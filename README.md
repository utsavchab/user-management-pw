# user-management-pw

## API Documentation
1) User Register
   - route: '/register'
   - method: 'POST'
   - keys = [name, email, password]

2) User Login
   - route: '/login'
   - method: 'POST'
   - keys = [email, password]

3) Environemnt Varaibles
   - PORT=PORTNUMBER
   - MONGO_URI='mongodb string'
